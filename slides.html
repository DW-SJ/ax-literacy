핵심 개념

에이전트 = 캐릭터

프롬프트를 매번 다시 쓰는 게 아닙니다.
역할을 한 번 정의해두면, 이름만 불러서 일을 시킵니다.

기존 방식
  • 새 대화 시작
  • 역할 설명 다시 입력
  • 프롬프트 붙여넣기
  • 결과 확인 → 끝
→
에이전트 방식
  • 캐릭터 파일 한 번 정의
  • 파일을 폴더에 넣기
  • /보고서요약 입력
  • 결과 파일이 자동 저장 — 오늘도, 내일도

# 웹사이트 만들기 요청

저는 대웅제약 AX팀에서 일하고 있습니다.

조건:
- 제목: "대웅제약 AX팀 — AI로 일하는 방식을 바꿉니다"
- 색상: 주황색(#F18A00) 포인트, 흰 배경
- 내용: 팀 소개, 주요 업무 3가지, 연락처 섹션

- HTML 파일 하나로 완성해주세요
📋 프롬프트 2 — 내용 수정 (원하는 대로 바꿔보세요)
# 수정 요청 — 원하는 부분을 자연스럽게 말하면 됩니다

좋아요! 몇 가지 바꿔주세요:
- 팀 소개 문구를 좀 더 임팩트 있게 수정해주세요
- 하단에 대웅제약 로고 이미지 자리를 만들어주세요
- 배경에 은은한 그라디언트 효과를 넣어주세요
- 전체적으로 더 현대적인 느낌으로 개선해주세요
✅ 결과 확인 방법
  • Claude가 HTML 코드 생성 → 코드 위 "Open in Browser" 버튼 클릭
  • 마음에 들 때까지 말로 수정 요청 가능. 코드는 볼 필요 없습니다.

  • 📄
    파일을 한 번 만들면 영구적으로 재사용
  • ✏️
    마음에 안 들면 파일만 수정하면 됨
  • ➕
    에이전트는 원하는 만큼 계속 추가 가능
📁 my-agents/
  📁 .claude/
    📁 commands/
