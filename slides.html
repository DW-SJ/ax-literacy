# 에이전트 만들기

.claude/commands/ 폴더 안에 보고분석.md 파일을 만들어줘.

이 파일은 내가 /보고분석 커맨드로 부를 에이전트야.
이 에이전트의 역할: 보고서를 읽고, Agent 툴로 전문가 3명을 동시에 실행해 결과를 종합하는 오케스트레이터

호출되면 이 순서대로 실행해:
1. 현재 폴더에서 .txt 파일을 찾아서 내용을 읽어
2. Agent 툴로 전문가 3명을 동시에 실행해 (각자에게 파일 내용을 전달):
   - 전문가 1 (즉시대응 분석가): 지금 당장 대응할 이슈 2가지 찾기
   - 전문가 2 (리스크 탐지자): 보고서에 없는 숨겨진 리스크 3가지 찾기
   - 전문가 3 (경영 자문가): CEO 질문 5개 + 말하지 않은 것 정리
3. 세 결과를 통합해서 보고분석_결과.html로 저장하고 브라우저에서 열어줘
⚙️ 파일이 만들어지면
  • 탐색기에 .claude/commands/보고분석.md 파일 생성 확인
  • 이제 다음 슬라이드에서 /보고분석 한 줄만 입력하면 됩니다

3. 세 결과를 받아 🎯 종합 판단을 추가하고 의사결정_분석.html로 저장하고 브라우저에서 열어줘

이 에이전트의 역할: Agent 툴로 리서처 4명을 동시에 실행해 오늘의 시장 동향을 실시간 수집하는 오케스트레이터

호출되면 이 순서대로 실행해:
1. Agent 툴로 리서처 4명을 동시에 실행해 (각자 WebSearch 사용):
   - Agent 1 (🏭 산업 리서처): 오늘 제약·바이오·헬스케어 주요 뉴스 3~5건
   - Agent 2 (📈 거시경제 분석가): 금리·환율·유가·주요 지수 오늘 동향
   - Agent 3 (🏢 경쟁사 모니터): 주요 제약사 신약·M&A·임상 결과 뉴스
   - Agent 4 (📋 규제 트래커): 식약처·FDA 최신 허가·규제 뉴스
2. 네 결과를 통합해서 시장브리핑_결과.html로 저장하고 브라우저에서 열어줘
