Step 1

VSCode 설치하기

VSCode는 마이크로소프트가 만든 무료 코드 편집기입니다. 우리는 코딩 도구가 아니라 Claude와 대화하는 공간으로 사용할 거예요.

1
VSCode 홈페이지 접속

브라우저 주소창에 code.visualstudio.com 을 입력하고 Enter

2
다운로드 버튼 클릭

파란색 "Download for Windows" 버튼을 클릭합니다

💡 자동으로 Windows용 파일이 선택됩니다
3
설치 파일 실행

다운로드된 VSCodeSetup-x64-xxx.exe 파일을 더블클릭 → "동의함" 체크 → Next를 계속 클릭 → Install

4
설치 완료 확인

설치가 끝나면 "Launch Visual Studio Code" 체크 후 Finish 클릭

✅ 파란 아이콘의 VSCode가 열리면 성공!
1
VSCode 홈페이지 접속

브라우저 주소창에 code.visualstudio.com 을 입력하고 Enter

2
Mac 버전 다운로드

"Download for Mac" 버튼 클릭 → .zip 파일이 다운로드됩니다

💡 Apple Silicon(M1/M2)이면 "Apple Silicon" 버전 선택
3
앱 설치

다운로드된 .zip 파일을 더블클릭하여 압축 해제 → Visual Studio Code.app 파일을 Applications 폴더로 드래그

4
앱 실행

Launchpad 또는 Applications 폴더에서 Visual Studio Code 클릭

✅ "개발자를 확인할 수 없음" 팝업이 뜨면 → 시스템 환경설정 → 보안 → "확인 없이 열기" 클릭
이미 설치되어 있다면? 바로 다음 단계로 넘어가세요. VSCode 버전은 최신이면 좋지만 크게 상관없습니다.

Step 2

VSCode 기본 화면 익히기

복잡해 보이지만, 오늘 우리가 사용할 부분은 딱 하나입니다. 왼쪽 사이드바의 Claude 아이콘만 기억하면 됩니다.

Visual Studio Code
✦
← 이 주황색 아이콘이 Claude Code입니다
클릭하면 오른쪽에 대화창이 열립니다.
여기서 Claude에게 한국어로 말하면 됩니다.
A
사이드바 (왼쪽)

Claude, 파일탐색기, 검색 등의 아이콘이 있습니다. 오늘은 Claude 아이콘만 씁니다.

B
편집창 (중앙)

파일 내용이 표시되는 공간. Claude가 코드나 문서를 생성하면 여기에 나타납니다.

C
터미널 (아래)

Claude가 자동으로 명령을 실행할 때 사용합니다. 직접 건드릴 필요는 없어요.

Step 3

Claude Code 확장 프로그램 연결하기

VSCode에 Claude를 연결하는 작업입니다. 딱 한 번만 하면 됩니다.

1
Extensions 패널 열기

Windows: Ctrl + Shift + X  /  Mac: ⌘ + Shift + X

또는 왼쪽 사이드바의 네모 아이콘(Extensions) 클릭

2
Claude Code 검색

검색창에 "Claude Code" 입력 → Anthropic이 만든 "Claude Code" 선택

💡 발행자가 "Anthropic"인 것을 골라야 합니다
3
Install 클릭

파란 Install 버튼을 클릭하면 자동으로 설치됩니다 (30초 이내)

4
로그인하기

설치 후 왼쪽 사이드바에 Claude 아이콘이 생깁니다 → 클릭 → "Sign in with Claude.ai" 선택 → 브라우저에서 로그인

✅ 로그인하면 VSCode 안에서 바로 대화할 수 있습니다
5
첫 대화 테스트

Claude 패널에서 "안녕하세요" 입력 후 Enter → Claude가 답하면 완료!

계정이 없다면? claude.ai 에서 이메일로 무료 가입 가능합니다. 회사 이메일(daewoong.co.kr)로 가입을 권장합니다.
