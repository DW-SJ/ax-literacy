대웅제약 AI 자동화 솔루션 도입 검토 안건
작성일: 2026년 6월 28일
검토부서: AX팀 / 경영기획팀
결재라인: 본부장 → 부사장 → 대표이사

═══════════════════════════════════════════
1. 검토 배경
═══════════════════════════════════════════

영업·마케팅 현장의 반복 업무(보고서 작성, 데이터 집계, 일정 조율)에
소요되는 시간이 팀 전체 업무의 약 38%에 달함 (2026년 1분기 업무분석 결과).
이를 AI 자동화로 전환할 경우 연간 약 2,400시간 절감 가능.

AI 자동화 솔루션 시장 조사 결과, 대웅제약 규모·업종에 적합한
후보 2개사로 압축하여 최종 의사결정 요청.

═══════════════════════════════════════════
2. 후보 솔루션 비교
═══════════════════════════════════════════

[A사 — AutoFlow Pro]
- 특징: 제약·바이오 특화 AI 플랫폼, 국내 대형 제약사 3곳 도입 완료
- 주요 기능: 영업보고 자동화, ERP 연동, 규제 문서 초안 생성
- 도입 비용: 초기 구축 1.8억 원 + 연간 라이선스 3.2억 원
- 도입 기간: 계약 후 약 5개월 (커스터마이징 포함)
- 레퍼런스: D제약, Y바이오, H헬스케어

[B사 — SmartOps AI]
- 특징: 범용 업무 자동화 플랫폼, 빠른 도입이 강점
- 주요 기능: 문서 자동화, 이메일 분류, 일정 관리 (제약 특화 기능 부재)
- 도입 비용: 초기 구축 0.6억 원 + 연간 라이선스 1.1억 원
- 도입 기간: 계약 후 약 6주
- 레퍼런스: 제조업·유통업 중심 (제약사 레퍼런스 없음)

═══════════════════════════════════════════
3. 주요 제약 조건
═══════════════════════════════════════════

[예산]
- 현재 승인된 AX 예산: 연간 1.5억 원
- 추가 예산 확보 가능 시점: 2026년 4분기 (예산 재편성 시)

[내부 역량]
- AX팀 인원: 6명 (IT 전문인력 2명 포함)
- 현재 병행 중인 프로젝트: Claude Code 내재화, ERP 고도화

[일정]
- 경영진 요청 사항: 2026년 내 가시적 성과 필요
- 3분기 주주총회 전 도입 방향 확정 권고

═══════════════════════════════════════════
4. 쟁점 사항
═══════════════════════════════════════════

① 예산 범위: 현재 예산으로는 B사만 가능, A사는 추가 확보 필요
② 속도 vs 완성도: B사는 6주 내 가동 가능하나 제약 특화 기능 부족
③ 확장성: A사는 향후 규제 문서 자동화까지 확장 가능, B사는 한계 있음
④ 리스크: A사 도입 실패 시 매몰비용 1.8억 원, B사는 0.6억 원

═══════════════════════════════════════════
5. 결정 요청 사항
═══════════════════════════════════════════

아래 세 가지 중 방향 결정 요청:

[Option 1] A사 즉시 계약
→ 추가 예산 1.7억 원 확보 방안 병행 마련

[Option 2] B사 우선 도입 → 4분기 A사 전환 검토
→ B사로 단기 성과 확보 후 예산 확보 시 A사 재검토

[Option 3] 현재 내부 구축(Claude Code 활용) 고도화 집중
→ 외부 솔루션 도입 보류, AX팀 자체 역량으로 자동화 추진
